Module 1 Portfolio

Setup:
1) Create and activate virtual environment
   python -m venv .venv
   source .venv/bin/activate  (Mac/Linux)

2) Install dependencies
   pip install -r requirements.txt

Run:
- Start the app with:
  $ python run.py

Notes:
- Templates are in app/templates
- Static assets are in app/static

Screenshots:
- screenshots/* contains screenshots of the running site.
